# Google Play Data Safety Draft

이 문서는 현재 코드 기준 초안입니다. 실제 제출 전에는 운영 백엔드와 콘솔 설정 기준으로 최종 검토해야 합니다.

## 데이터 유형 후보

- 개인 정보
  - 이메일 주소
  - 이름 또는 대표자명
  - 전화번호
  - 주소
- 앱 활동
  - 앱 내 상호작용
  - 업로드한 이미지 및 OCR 처리 데이터
- 앱 정보 및 성능
  - 오류 로그
  - 진단 정보

## 수집 목적 후보

- 앱 기능 제공
  - 로그인, 회원 식별, 매장별 데이터 분리
  - 매출/재고/레시피 관리
  - OCR 이미지 분석
- 고객지원
  - 계정 문제 대응
  - 삭제 요청 처리
- 보안 및 사기 방지
  - 비정상 접근 탐지
  - 감사 로그

## 공유 여부 검토 포인트

- 인증 제공자와의 처리 여부
- 클라우드 호스팅 및 저장소 제공자와의 처리 여부
- OCR/AI 처리 공급자와의 처리 여부
- 광고 목적 제3자 공유는 현재 코드 기준 전제하지 않음

## 삭제 정책

- 앱 내 계정 삭제 요청 경로 제공
- 공개 삭제 URL 제공
- 법적 보존 의무 또는 보안 사유가 있는 데이터는 즉시 삭제되지 않을 수 있음
